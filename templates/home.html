Modul Analisis Inti

Network & Geo-Intelligence

Melacak alamat IP asli server, lokasi geografis, ISP, hingga mendeteksi *Tech Stack* dan CMS yang digunakan target.

Vulnerability Mitigation

Mendeteksi hilangnya HTTP Security Headers (HSTS, CSP) dan memberikan panduan mitigasi langsung dari dokumentasi OWASP/MDN.

Threat & Malware Reputation

Terintegrasi langsung dengan API VirusTotal untuk memastikan domain target bersih dari database malware dan indikasi phishing.

Advanced OSINT Discovery

Mencari subdomain tersembunyi menggunakan Certificate Transparency Logs (crt.sh) dan melacak data WHOIS registrar.

Open Ports & Sensitive Files

Pemindai port super cepat (FTP, SSH, Database) dan pelacak file terekspos seperti /.env atau /robots.txt.

Email Spoofing Check (DNS)

Menganalisis DNS Records (SPF & DMARC) untuk melihat apakah domain rentan digunakan untuk penipuan email spoofing.
