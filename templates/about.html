Dokumentasi & Arsitektur

Spesifikasi teknis dan regulasi penggunaan WebShield Analyzer.

WebShield Analyzer dibangun menggunakan arsitektur Python/Flask modern, dikombinasikan dengan teknik intelijen siber (Cyber Intelligence) berbasis API dan HTTP Request.

Metodologi Analisis

OSINT Pasif

Mengumpulkan data dari arsip publik (crt.sh, VirusTotal, Wayback Machine, WHOIS) tanpa melakukan interaksi langsung yang dicurigai oleh firewall server target.

Scanning Aktif

Melakukan request HTTP standar untuk membaca struktur header, mendeteksi CMS, dan memindai port terbuka dengan toleransi timeout 0.5s untuk efisiensi.
